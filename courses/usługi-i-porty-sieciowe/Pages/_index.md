## Moduł 1: Usługi infrastrukturalne

# DNS - port 53 - tłumaczenie nazw
# DHCP - porty 67 i 68 - automatyczne IP
# NTP - port 123 - synchronizacja czasu
# SNMP - porty 161 i 162 - monitoring sieci
# LDAP - port 389 - katalog użytkowników

## Moduł 2: Usługi transferu danych

# HTTP i HTTPS - porty 80 i 443 - sieć WWW
# FTP i SFTP - porty 21 i 22 - transfer plików
# SMB i CIFS - porty 445 i 139 - udziały Windows
# NFS - port 2049 - udziały Linux

## Moduł 3: Usługi komunikacyjne

# SMTP - port 25 i 587 - wysyłanie poczty
# POP3 i IMAP - porty 110 i 143 - odbieranie poczty
# SSH - port 22 - bezpieczny zdalny dostęp
# RDP - port 3389 - Pulpit zdalny Windows
# VPN - protokoły i porty tunelowania
