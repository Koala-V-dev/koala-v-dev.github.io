# Równania liniowe

# 📘 **Komplet praw i twierdzeń potrzebnych do równań liniowych**

## 🎯 **1. Fundament: co to jest równanie liniowe**
Równanie liniowe ma postać:
\[
ax + b = c
\]
gdzie \(a, b, c \in \mathbb{R}\) i \(a \neq 0\).

Aby je rozwiązać, musisz umieć wykonywać działania na liczbach rzeczywistych i stosować prawa przekształcania równań.

---

# 🧩 **2. Prawa działań na równaniach (kluczowe)**

To są **jedyny zestaw praw**, które wolno stosować do obu stron równania. Każde z nich wymaga określonych umiejętności.

---

## **2.1. Prawo dodawania do obu stron**
\[
a = b \quad \Longrightarrow \quad a + c = b + c
\]

**Wymagane umiejętności:**  
- dodawanie liczb całkowitych i wymiernych  
- redukcja wyrazów podobnych  

---

## **2.2. Prawo odejmowania od obu stron**
\[
a = b \quad \Longrightarrow \quad a - c = b - c
\]

**Wymagane:**  
- działania na liczbach ujemnych  
- przekształcanie wyrażeń algebraicznych  

---

## **2.3. Prawo mnożenia obu stron przez tę samą liczbę**
\[
a = b \quad \Longrightarrow \quad ac = bc
\]
z zastrzeżeniem: \(c \neq 0\) **nie jest wymagane** (mnożyć można przez 0, tylko nie dzielić).

**Wymagane:**  
- mnożenie ułamków  
- mnożenie liczb ujemnych  

---

## **2.4. Prawo dzielenia obu stron przez tę samą liczbę**
\[
a = b \quad \Longrightarrow \quad \frac{a}{c} = \frac{b}{c}, \quad c \neq 0
\]

**Wymagane:**  
- dzielenie ułamków  
- rozumienie zakazu dzielenia przez 0  

---

## **2.5. Prawo przenoszenia wyrazów na drugą stronę**
To tylko skrót myślowy wynikający z 2.1 i 2.2:
\[
ax + b = c \quad \Longrightarrow \quad ax = c - b
\]

**Wymagane:**  
- rozumienie, że „przenoszenie” = dodanie/odjęcie przeciwnych wartości  

---

## **2.6. Prawo redukcji wyrazów podobnych**
\[
3x + 5x = 8x
\]

**Wymagane:**  
- pojęcie współczynnika  
- działania na liczbach  

---

## **2.7. Prawo rozdzielności (dystrybutywność)**
\[
a(b + c) = ab + ac
\]

**Wymagane:**  
- mnożenie liczb  
- umiejętność wyłączania wspólnego czynnika  

---

## **2.8. Prawo łączenia (asocjacyjność)**
\[
(a + b) + c = a + (b + c)
\]

**Wymagane:**  
- podstawowe działania  

---

## **2.9. Prawo przemienności**
\[
a + b = b + a
\]

**Wymagane:**  
- podstawowe działania  

---

# 🧮 **3. Wymagania matematyczne (co trzeba umieć wcześniej)**

## **3.1. Liczby i działania**
- liczby całkowite, wymierne, rzeczywiste  
- ułamki zwykłe i dziesiętne  
- działania na liczbach ujemnych  
- porównywanie liczb  

---

## **3.2. Algebra elementarna**
- redukcja wyrazów podobnych  
- wyłączanie wspólnego czynnika  
- rozdzielność mnożenia względem dodawania  
- przekształcanie prostych wyrażeń  

---

## **3.3. Potęgi i pierwiastki (opcjonalnie, gdy równanie zawiera takie elementy)**
- potęgi o wykładniku naturalnym  
- pierwiastki drugiego stopnia  
- własności:  
  \[
  \sqrt{a^2} = |a|,\quad a^m \cdot a^n = a^{m+n}
  \]

---

## **3.4. Ułamki algebraiczne**
- skracanie  
- sprowadzanie do wspólnego mianownika  
- dzielenie ułamków  

---

## **3.5. Logika matematyczna (minimalny zestaw)**
- pojęcie równoważności:  
  \[
  A \iff B
  \]
- pojęcie implikacji:  
  \[
  A \Rightarrow B
  \]
- rozumienie, że równanie to **predykat**, a nie „polecenie obliczenia”

**Prawo de Morgana NIE jest potrzebne** do równań liniowych — przydaje się dopiero w logice zdań i zbiorach.

---

# 🧭 **4. Twierdzenia i fakty o równaniach liniowych**

## **4.1. Równanie liniowe ma dokładnie jedno rozwiązanie**
Jeśli \(a \neq 0\), to:
\[
ax + b = c \quad \Longrightarrow \quad x = \frac{c - b}{a}
\]

---

## **4.2. Równanie sprzeczne**
\[
ax + b = ax + c,\quad b \neq c
\]
→ brak rozwiązań.

---

## **4.3. Równanie tożsamościowe**
\[
ax + b = ax + b
\]
→ nieskończenie wiele rozwiązań.

---

## **4.4. Równania równoważne**
Dwa równania są równoważne, jeśli mają ten sam zbiór rozwiązań.

---

## **4.5. Zasada zachowania równoważności**
Każde działanie wykonane na obu stronach równania musi zachować prawdziwość równania.

---

# 🧱 **6. Minimalny pakiet wiedzy ucznia (checklista)**

Uczeń musi umieć:

- dodawać, odejmować, mnożyć i dzielić liczby (w tym ułamki)  
- wykonywać działania na liczbach ujemnych  
- redukować wyrazy podobne  
- stosować rozdzielność  
- przenosić wyrazy na drugą stronę  
- dzielić przez współczynnik przy \(x\)  
- sprawdzać rozwiązanie

# 📘 **TWIERDZENIA O RÓWNANIACH LINIOWYCH (pełna lista)**

## 1. **Twierdzenie o jednoznaczności rozwiązania równania liniowego**
Jeśli \(a \neq 0\), to równanie:
\[
ax + b = c
\]
ma **dokładnie jedno rozwiązanie**:
\[
x = \frac{c - b}{a}
\]

**Wymagania:** ułamki, dzielenie, działania na liczbach rzeczywistych.

---

## 2. **Twierdzenie o równaniu sprzecznym**
Równanie liniowe nie ma rozwiązań wtedy i tylko wtedy, gdy:
\[
ax + b = ax + c \quad \text{oraz} \quad b \neq c
\]

**Interpretacja:** lewa i prawa strona mają ten sam współczynnik przy \(x\), ale różne wyrazy wolne.

**Wymagania:** redukcja wyrazów podobnych.

---

## 3. **Twierdzenie o równaniu tożsamościowym**
Równanie ma **nieskończenie wiele rozwiązań**, gdy:
\[
ax + b = ax + b
\]

**Wymagania:** rozumienie równoważności równań.

---

## 4. **Twierdzenie o równoważności równań**
Dwa równania są równoważne wtedy i tylko wtedy, gdy mają **ten sam zbiór rozwiązań**.

To jest fundament całej teorii przekształceń.

**Wymagania:** logika matematyczna (implikacja, równoważność).

---

## 5. **Twierdzenie o zachowaniu równoważności przy działaniach elementarnych**
Jeśli wykonasz na obu stronach równania jedno z działań:

- dodanie tej samej liczby  
- odjęcie tej samej liczby  
- pomnożenie przez tę samą liczbę  
- podzielenie przez tę samą liczbę (≠ 0)

to otrzymane równanie jest **równoważne** wyjściowemu.

Formalnie:
\[
a=b \iff a+c=b+c
\]
\[
a=b \iff a-c=b-c
\]
\[
a=b \iff ac=bc
\]
\[
a=b \iff \frac{a}{c}=\frac{b}{c},\; c\neq 0
\]

**Wymagania:** działania na liczbach, ułamki, logika.

---

## 6. **Twierdzenie o istnieniu i jednoznaczności rozwiązania układu jednego równania liniowego**
Układ:
\[
ax + b = c
\]
jest układem **oznaczonym** (ma jedno rozwiązanie), jeśli \(a \neq 0\).

To jest wersja „układowa” twierdzenia nr 1.

---

## 7. **Twierdzenie o równaniach liniowych jako funkcjach**
Równanie:
\[
ax + b = c
\]
jest równoważne przecięciu wykresów dwóch funkcji liniowych:
\[
y_1 = ax + b,\quad y_2 = c
\]

Rozwiązanie to punkt przecięcia prostych.

**Wymagania:** funkcja liniowa, wykresy.

---

## 8. **Twierdzenie o monotoniczności funkcji liniowej**
Jeśli \(a > 0\), funkcja \(ax + b\) jest rosnąca.  
Jeśli \(a < 0\), jest malejąca.

Z tego wynika, że równanie liniowe ma **zawsze jedno rozwiązanie**, bo funkcja monotoniczna przecina prostą poziomą dokładnie raz.

**Wymagania:** funkcje, monotoniczność.

---

## 9. **Twierdzenie o przekształcaniu równań liniowych do postaci kanonicznej**
Każde równanie liniowe jednej zmiennej można sprowadzić do postaci:
\[
x = d
\]
lub do sprzeczności:
\[
0 = k,\; k \neq 0
\]
lub do tożsamości:
\[
0 = 0
\]

**Wymagania:** redukcja, działania elementarne.

---

## 10. **Twierdzenie o proporcjonalności**
Jeśli:
\[
ax = c
\]
to:
\[
x = \frac{c}{a}
\]

To jest fundament skalowania, proporcji i przekształceń liniowych.

**Wymagania:** ułamki, dzielenie.

---

# 🔥 **BONUS: Twierdzenia, które często są mylone z „prawami równań”, ale nimi nie są**

**Prawo de Morgana nie ma zastosowania do równań liniowych.**
