NAZWA
    ufw - prosta zapora systemowa dla Ubuntu

SKŁADNIA
    sudo ufw status
    sudo ufw enable
    sudo ufw disable
    sudo ufw allow PORT/PROTO
    sudo ufw deny PORT/PROTO
    sudo ufw delete allow PORT/PROTO

OPIS
    ufw jest wygodną nakładką na reguły zapory. W kursie Desktop używamy go do zrozumienia, że zapora reguluje ruch przychodzący do usług działających na komputerze.

PRZYKŁADY
    sudo ufw status
    sudo ufw enable
    sudo ufw allow 22/tcp
    sudo ufw delete allow 22/tcp

UWAGI
    Pełne reguły usług, SSH i ekspozycja portów będą rozwinięte w kursie Ubuntu Server.
