Wizualizator Kaskady i Obliczania Specyficzności CSS
Koala-V

Reguły w arkuszu stylów CSS (Klikaj aby włączać/wyłączać)

p Waga: (0,0,0,1) = 1
color: #60a5fa; /* Kolumna D - Element */
.card p Waga: (0,0,1,1) = 11
color: #34d399; /* Kolumny C+D - Klasa + Element */
#main-content .active:hover Waga: (0,1,2,0) = 120
color: #a855f7; /* Kolumny B+C - ID + Klasy */
style="..." Waga: (1,0,0,0) = 1000
color: #f43f5e; /* Kolumna A - Styl wbudowany */

Wynik renderowania w przeglądarce

<div id="main-content" class="card">
  <p class="active"> Tekst testowy   </p>
</div>

Kalkulator Kaskady

p (Element) (0, 0, 0, 1) = 1
.card p (Klasa + Element) (0, 0, 1, 1) = 11
#main-content .active:hover (ID + Klasy) (0, 1, 2, 0) = 120
style="..." (Styl wbudowany) (1, 0, 0, 0) = 1000
Klikaj w karty reguł CSS po lewej stronie, aby zobaczyć w czasie rzeczywistym, jak wyłączenie lub włączenie reguły zmienia kolor elementu HTML i jak obliczana jest waga zwycięzcy.
